crear un .php con la conexion, para incluir en las paginas:

<?php
$host = "localhost";
$user = "root";
$pass = ""; // la que se haya creado 
$db   = "UPEGaming";

$conn = new mysqli($host, $user, $pass, $db);

if ($conn->connect_error) {
    die("Error de conexion: " . $conn->connect_error);
}
?>
